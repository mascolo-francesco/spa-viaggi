# Frontend Responsibilities (Handoff)

## Ownership
La responsabilita completa del frontend e assegnata a `claude-sonnet-4.6`.

## Stack e strumenti richiesti
- Framework: `React`.
- Skill principale: `frontend-design`.
- UI components: `shadcn/ui` con skill dedicata.
- MCP opzionale: shadcn MCP per retrieval componenti quando necessario.

## Obiettivo qualitativo
Frontend professionale, coerente con i contratti API backend e pronto per demo/valutazione orale.

## Boundary con backend
- Backend fornisce API, validazioni, error model e OpenAPI.
- Frontend gestisce rendering, stato UI, navigazione, form, UX operativa.
- Nessuna logica business critica duplicata lato frontend.
